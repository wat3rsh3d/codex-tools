- [x] Skills-only plugin manifest

- [x] Skill-bundle and plugin-bundle build script
- [x] Public name, short description, and long description
- [x] Production SVG logo
- [x] Starter prompts
- [x] Exactly five positive test cases

- [x] Initial release notes
